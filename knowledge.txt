The SQL CREATE statement is used to create new objects in a database.

Common uses of CREATE:
1. CREATE DATABASE - Creates a new database.
   Example:
   CREATE DATABASE school;

2. CREATE TABLE - Creates a new table in a database.
   Example:
   CREATE TABLE students (
       id INT PRIMARY KEY AUTO_INCREMENT,
       name VARCHAR(100),
       age INT,
       grade VARCHAR(10)
   );

3. CREATE USER - Creates a new user for the database system.
   Example:
   CREATE USER 'pushpendra'@'localhost' IDENTIFIED BY 'password123';

4. CREATE INDEX - Creates an index (for faster searching).
   Example:
   CREATE INDEX idx_name ON students(name);

Summary:
The CREATE statement defines new structures (databases, tables, users, etc.) that store and organize data.
